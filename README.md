# tood

## Local Development

```
npm install \
  && chmod +x tood
  && ./tood --version
```

## Usage

### help

Show help

```
./tood help
```

### create

Create a tood item with description

```
./tood create "Do the washing"
```
### list

List all toods

```
./tood list
```

### remove

Remove a todo by id

```
./tood remove 0
```
